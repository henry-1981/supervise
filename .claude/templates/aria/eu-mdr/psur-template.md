## Template Usage Notes

1. **Customization Instructions**:
   - Adapt for device class
   - Add device-specific sections
   - Include all appendices

2. **Reporting Frequency**:
   - Class I devices: Not routinely required (unless requested)
   - Class IIa devices: Every 2-3 years
   - Class IIb devices: Every 2 years
   - Class III devices: Annually

3. **Content Requirements**:
   - Be comprehensive and accurate
   - Include all relevant safety data
   - Provide clear trend analysis
   - Document all significant changes
