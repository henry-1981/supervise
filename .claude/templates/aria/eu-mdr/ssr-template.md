## Template Usage Notes

1. **Customization Instructions**:
   - Adapt for device type and class
   - Include all device variants
   - Translate into official language of Member State

2. **Content Requirements**:
   - Be concise and clear
   - Focus on safety and performance
   - Use plain language where possible
   - Include all required sections

3. **Update Requirements**:
   - Update when significant changes occur
   - Update at least every 2-3 years (or annually for Class III)
   - Update when new safety information becomes available

4. **Publication**:
   - SSR must be published in EUDAMED
   - SSR must be freely accessible
   - SSR must be up-to-date
