# Device Approval (기기 허가)

## 품목신고 (Item Notification)

### 대상
- Class I 기기 대부분
- 일부 Class II 기기

### 절차
1. 기술 문서 작성
2. 품목신고서 제출
3. MFDS 접수 확인
4. 신고증 발급
5. 시장 출시

### 서류
- 품목신고서
- 기술 문서
- 사용 설명서
- 라벨

## 품목허가 (Item Approval)

### 대상
- Class II, III, IV 기기

### 절차
1. 기술 문서 작성
2. 임상시험 (필요 시)
3. 품목허가신청서 제출
4. MFDS 심사
5. 품목허가증 발급
6. 시장 출시

### 서류
- 품목허가신청서
- 기술 문서
- 안전성·유효성 자료
- 임상시험 자료 (필요 시)
- 사용 설명서
- 라벨

## 심사 기간
- Class II: 60일
- Class III: 90일
- Class IV: 180일
