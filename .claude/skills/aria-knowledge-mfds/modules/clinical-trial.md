# Clinical Trial (임상시험)

## 임상시험 승인 절차

### 1. 임상시험계획서 작성
- 임상시험 목적
- 임상시험 설계
- 피험자 선정 기준
- 평가 항목
- 통계 계획

### 2. IRB 승인
- 임상시험계획서 제출
- IRB 심의
- 승인 획득

### 3. MFDS 승인
- 임상시험 승인 신청서
- 임상시험계획서
- IRB 승인서
- 피험자 동의서
- MFDS 심사 (30일)

### 4. 임상시험 실시
- 피험자 모집
- 임상시험 수행
- 데이터 수집

### 5. 결과 보고
- 임상시험 결과서
- MFDS 보고

## 임상시험 면제
- 동일 기기 임상 데이터
- 문헌 검토
- 기존 허가 기기 비교
- Post-Market Surveillance
