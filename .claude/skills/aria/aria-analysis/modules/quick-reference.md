# ARIA Analysis Quick Reference

## Analysis Process

1. **Define**: Clear objectives and questions
2. **Prepare**: Collect and clean data
3. **Analyze**: Apply appropriate methods
4. **Interpret**: Statistical and practical significance
5. **Communicate**: Clear visualizations and reports

## Common Statistical Tests

| Test | Purpose | When to Use |
|------|---------|-------------|
| t-test | Compare 2 group means | Continuous outcome, 2 groups |
| ANOVA | Compare 3+ group means | Continuous outcome, 3+ groups |
| Chi-square | Categorical associations | Categorical data |
| Correlation | Variable relationships | Continuous variables |
| Regression | Model/predict | Prediction, explanation |

## Chart Selection

| Purpose | Chart Type |
|---------|------------|
| Comparison | Bar chart |
| Trend | Line chart |
| Distribution | Histogram |
| Relationship | Scatter plot |
| Composition | Pie/Stacked bar |

## Data Quality Dimensions

- Completeness: All required fields present
- Accuracy: Values within expected ranges
- Consistency: No conflicting values
- Timeliness: Data is current
- Validity: Data types correct
