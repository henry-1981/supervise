# Web Research

## Overview

Systematic web research for gathering information and best practices.

## Research Process

### 1. Query Formulation

- Define research objectives
- Identify key search terms
- Formulate targeted queries
- Plan search strategy

### 2. Information Gathering

- Use WebSearch for initial research
- Apply WebFetch for URL verification
- Document findings with sources
- Cross-reference information

### 3. Source Evaluation

- Verify URL accessibility
- Check source credibility
- Validate information accuracy
- Document confidence level

### 4. Knowledge Synthesis

- Consolidate findings
- Identify patterns and trends
- Extract actionable insights
- Document recommendations

## Best Practices

- Always verify URLs before including
- Include Sources section when WebSearch used
- Use multiple sources for validation
- Document uncertainty when information unclear
