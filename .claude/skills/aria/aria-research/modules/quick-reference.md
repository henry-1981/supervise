# ARIA Research Quick Reference

## SOURCE Criteria

| Criterion | Description | Check |
|-----------|-------------|-------|
| **S**tandard | Official standard/regulation? | Yes/No |
| **O**riginal | From authoritative body? | Yes/No |
| **U**pdated | Current version? | Yes/No |
| **R**elevant | Addresses research question? | Yes/No |
| **C**omplete | Information complete? | Yes/No |
| **E**xplicit | Clear and unambiguous? | Yes/No |

## Citation Format Examples

**Regulatory**: FDA, 21 CFR Part 820, Quality System Regulation, 2024.
**Standard**: ISO, ISO 13485:2016, Medical devices - Quality management systems, 2016.
**Guidance**: FDA, "Guidance for Industry: Q8(R2) Pharmaceutical Development," 2009.

## Research Process

1. **Plan**: Define scope, identify sources
2. **Gather**: Collect from authoritative sources
3. **Verify**: Cross-reference, check currency
4. **Document**: Maintain source trail, cite properly

## Quality Dimensions

- Accuracy: Correct and factual
- Currency: Current and up-to-date
- Relevance: Addresses research question
- Authority: From credible sources
- Completeness: Comprehensive coverage
