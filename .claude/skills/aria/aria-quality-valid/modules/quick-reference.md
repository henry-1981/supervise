# VALID Framework Quick Reference

## Five Dimensions at a Glance

| Dimension | Focus | Key Check | Max Score |
|-----------|-------|-----------|-----------|
| **V**erified | Content matches original regulations | Cross-reference with source | 20 |
| **A**ccurate | Data/numbers/refs are correct | Source verification | 20 |
| **L**inked | Requirements-docs-evidence traceability | Traceability matrix | 20 |
| **I**nspectable | Audit trails with rationale | Decision documentation | 20 |
| **D**eliverable | Meets submission format | Template compliance | 20 |

## Quality Levels

- **Excellent** (90-100): Ready for submission
- **Good** (70-89): Minor improvements needed
- **Acceptable** (50-69): Significant improvements needed
- **Poor** (0-49): Major revisions required

## Quick Checklist

- [ ] Regulatory text cross-referenced
- [ ] All claims have verifiable sources
- [ ] Requirements mapped to sections
- [ ] Decisions have documented rationale
- [ ] Template requirements satisfied
