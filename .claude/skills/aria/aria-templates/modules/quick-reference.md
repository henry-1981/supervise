# ARIA Templates Quick Reference

## Template Categories

| Category | Templates | Use Case |
|----------|-----------|----------|
| Regulatory | Submission, Assessment | Formal submissions, compliance |
| Technical | SOP, Technical Report | Procedures, analysis |
| Review | Checklist, Review Comments | Peer review, quality review |
| Project | Status Report, Meeting Minutes | Progress tracking, meetings |

## Standard Template Elements

All ARIA templates include:
- Document control (ID, version, date, owner)
- Revision history
- Required sections for document type
- VALID framework compliance

## Template Selection Guide

1. Identify document purpose
2. Determine audience
3. Check regulatory requirements
4. Choose matching template

\`\`\`markdown

[Standard sections for document type]
