# ARIA Writing Style Quick Reference

## Core Principles

1. **Clarity First**: Write for your audience, use simple language
2. **Consistency**: Standardized terminology and formatting
3. **Accuracy**: Verify all claims, cite sources
4. **Completeness**: Cover all required topics

## Key Formatting Rules

| Element | Format | Example |
|---------|--------|---------|
| Dates | ISO 8601 | 2025-02-09 |
| Times | 24-hour with timezone | 14:30 KST |
| Numbers 0-10 | Spelled out | three requirements |
| Numbers 11+ | Figures | 25 documents |
| Units | Space before | 10 MB |
| Code | Monospace backticks | `variable` |
| Emphasis | Bold asterisks | **important** |

## Heading Hierarchy

```markdown
# H1: Document Title (use only once)
## H2: Major Sections
### H3: Subsections
```

## Sentence Guidelines

- **Use**: Subject-Verb-Object, active voice, present tense
- **Avoid**: Passive voice, complex sentences, 25+ words

## Quick Checklist

- [ ] Spelling and grammar checked
- [ ] Consistent terminology
- [ ] Active voice used
- [ ] Acronyms defined first
- [ ] Dates in ISO format
- [ ] Template requirements met
