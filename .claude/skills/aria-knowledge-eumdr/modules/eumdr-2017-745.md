# EU MDR 2017/745

## Overview

EU MDR (Medical Device Regulation) 2017/745은 유럽 연합의 의료기기 규제를 담당하는 법령입니다. 2017년 5월 5일에 제정되었으며, 2021년 5월 26일부터 완전히 적용되었습니다. 기존 MDD (Medical Device Directive) 93/42/EEC를 대체합니다.

## Key Changes from MDD

1. **Scope Expansion**:
   - 의료기기 정의 확대 ( aesthetic devices 포함)
   - 소프트웨어 의료기기 명시적 포함

2. **Classification Rules**:
   - Rule 10-22 추가 (총 22개 규칙)
   - 분류 기준 강화

3. **Notified Body**:
   - 더 엄격한 지정 기준
   - Unannounced audits 도입

## Device Classification

### Class I
- **Definition**: 저위험 의료기기
- **Examples**:
  - Non-invasive devices (예: 안경, 의자형 휠체어)
  - Invasive devices (예: 재사용 가능한 수술 도구)
  - Active therapeutic devices (예: 저주파 치료기)
- **Requirements**:
  - 자체 선언 (Self-certification)
  - QMS 요구
  - 기술 문서 작성

### Class IIa
- **Definition**: 중간 위험 의료기기
- **Examples**:
  - 치과 임플란트
  - 혈액 투석기
  - X-ray 진단기기
- **Requirements**:
  - Notified Body 참여 필요
  - QMS 심사
  - 기술 문서 검토

### Class IIb
- **Definition**: 높은 위험 의료기기
- **Examples**:
  - 스텐트
  - 인공관절
  - 장기 이식용 의료기기
- **Requirements**:
  - Notified Body 전체 심사
  - 임상 평가 필요
  - 기술 문서 상세 검토

### Class III
- **Definition**: 최고 위험 의료기기
- **Examples**:
  - 심장박동조절기
  - 뇌 이식 장치
  - 혈관 내 코일/스텐트
- **Requirements**:
  - Notified Body 전체 심사
  - 임상 시험 필요
  - PMS (Post-Market Surveillance) 강화

## Technical Documentation

### Required Contents

1. **Device Description**
   - Intended purpose
   - Product variants
   - Accessories

2. **Safety & Performance**
   - General safety and performance requirements (Annex I)
   - Benefit-risk ratio
   - Clinical evaluation

3. **Design & Manufacturing**
   - Design validation
   - Manufacturing processes
   - Sterilization (if applicable)

4. **Labeling**
   - UDI (Unique Device Identification)
   - IFU (Instructions for Use)
   - CE marking

## Clinical Evaluation

### MEDDEV 2.7/1 Rev 4

**Clinical Evaluation Report (CER) 포함:**
1. Clinical data gathering
2. Applicability determination
3. Data analysis
4. Safety & performance demonstration

**Clinical Investigation:**
- Class III: 필수
- Class IIb: 대부분 필요
- Class IIa/I: 일부 경우 필요

## UDI Requirements

### UDI-DI (Device Identifier)
- 제조사별 고유 식별자
- Labeling 포함

### UDI-PI (Production Identifier)
- Lot/Batch number
- Serial number
- Expiration date
- Manufacturing date

### EUDAMED Database
- UDI 등록
- Economic Operators registration
- Clinical Investigation registration

## Quality Management

### ISO 13485:2016 Alignment

MDR Annex IX에 따른 QMS 요구사항:

1. **Management Responsibility**
   - Quality policy
   - Quality objectives
   - Management review

2. **Resource Management**
   - Competent personnel
   - Infrastructure
   - Work environment

3. **Product Realization**
   - Design controls
   - Purchasing controls
   - Production controls
   - Identification & traceability

4. **Measurement, Analysis, Improvement**
   - Monitoring & measurement
   - Nonconforming product
   - Corrective & preventive action

## Post-Market Surveillance (PMS)

### PMS Plan

**필수 요소:**
1. Proactive data collection
2. Complaint handling
3. Trend reporting
4. Field Safety Corrective Actions (FSCA)
5. Periodic Safety Update Report (PSUR)

### PSUR Requirements

- **Class III**: 연간 PSUR
- **Class IIb**: 연간 PSUR (일부 예외)
- **Class IIa/I**: 요청 시 PSUR
