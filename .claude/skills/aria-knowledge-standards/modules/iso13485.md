# ISO 13485:2016 - Quality Management System

의료기기 품질 경영 시스템 표준.

## ISO 13485:2016 개요

**Purpose:** 의료기기 제조사를 위한 QMS 요구사항

**Scope:** 의료기기 전체 수명 주기

**Key Changes from ISO 13485:2012**
- Risk-based approach 강화
- Regulatory focus 강화
- Supply chain 관리 강화
- Design control 명시

### Design Verification vs Validation

| 항목 | Verification | Validation |
|------|-------------|------------|
| 목적 | "올바른 제품을 만드는가?" | "올바른 제품을 만드는가?" |
| 대상 | Design Output | Design Input |
| 방법 | Testing, inspection | Clinical trials |
| 질문 | Did we build it right? | Did we build the right thing? |

## ISO 13485 vs ISO 9001

| 항목 | ISO 13485:2016 | ISO 9001:2015 |
|------|-----------------|---------------|
| Focus | Medical devices | General quality |
| Risk | Explicit requirement | Implicit |
| Regulatory | Regulatory focus | Customer focus |
| Design | Detailed requirements | General requirements |
| Validation | Specific requirements | Less specific |
