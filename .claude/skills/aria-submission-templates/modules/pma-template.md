# FDA PMA Template

## 필수 구성 요소

### 1. Administrative
- Form FDA 3514
- Cover Letter
- User Fee Cover Sheet
- Indications for Use Statement

### 2. Nonclinical Studies
- Bench Testing
- Animal Studies
- Engineering Evaluation
- Biocompatibility (ISO 10993)
- Electrical Safety (IEC 60601-1)
- EMC (IEC 60601-1-2)
- Software Validation (IEC 62304)
- Usability Engineering (IEC 62366-1)

### 3. Clinical Investigation
- Clinical Study Protocol
- Clinical Study Reports
- Statistical Analysis
- Clinical Data Summary

### 4. Manufacturing
- Manufacturing Process
- Quality System
- Facility Information
- Process Validation
- Sterilization Validation

### 5. Labeling
- Device Labeling
- Instructions for Use
- Patient Labeling
- Package Insert

## PMA Process
1. Pre-IDE Meeting
2. IDE Approval
3. PMA Submission
4. Advisory Panel (필요 시)
5. FDA Review (180일)
6. Approval Decision
