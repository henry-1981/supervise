# Korea MFDS Device Approval

## 필수 구성 요소

### 1. 행정 서류
- 품목허가신청서
- 사업자등록증
- 시설 등록증
- 위임장 (필요 시)

### 2. 제품 정보
- 기기 명칭 및 형번
- 제조업체 정보
- 제조 국가
- 모델명 및 식별번호

### 3. 제품 설명
- 기기 설명
- 작동 원리
- 구성 요소
- 첨부 기기

### 4. 의도된 사용
- 의도된 사용 목적
- 적응증
- 금기사항
- 사용상 주의사항

### 5. 안전성·유효성 자료
- 비임상시험 자료
- 임상시험 자료 (필요 시)
- 문헌 고찰

### 6. 위험 관리 자료
- 위험 분석
- 위험 평가
- 위험 경감 조치

### 7. 사용 설명서
- 사용 설명서 (한글)
- 라벨 (한글)
- 포장지 표시

### 8. 품질 경영 시스템
- K-GMP 인증서
- 품질 매뉴얼
- 제조 및 검사 규격

## 심사 기간
- Class II: 60일
- Class III: 90일
- Class IV: 180일
