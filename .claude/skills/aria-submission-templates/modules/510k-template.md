# FDA 510(k) Template

## 필수 구성 요소

### 1. Administrative
- Cover Letter
- User Fee Cover Sheet (Form FDA 3601)
- Indications for Use Statement
- 510(k) Statement
- Truthful and Correct Statement

### 2. Device Information
- Device Description
- Proposed Labeling
- Classification Information (21 CFR Part, Product Code)

### 3. Substantial Equivalence
- Predicate Device Identification
- Comparison to Predicate
- Differences and Similarities

### 4. Testing Data
- Performance Testing
- Biocompatibility (ISO 10993)
- Electrical Safety (IEC 60601-1)
- EMC (IEC 60601-1-2)
- Software (IEC 62304)
- Usability (IEC 62366-1)
- Sterilization Validation
- Shelf Life Testing

### 5. Risk Analysis
- Risk Management File (ISO 14971)
- Risk-Benefit Analysis

## 510(k) Types

### Traditional 510(k)
- 가장 일반적인 유형
- 모든 SE 항목 입증

### Abbreviated 510(k)
- Guidance Document 활용
- 표준 준수로 간소화

### Special 510(k)
- 기존 510(k) 승인 기기의 변경
- 30일 순환 목표
