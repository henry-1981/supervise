# Clinical Evaluation Overview (임상 평가 개요)

## 개요

Clinical Evaluation은 의료기기의 안전성과 유효성을 임상 데이터를 통해 입증하는 프로세스입니다.

## 임상 평가 방법

### 1. Literature Review (문헌 고찰)
- 기존 문헌 검색
- 과거 연구 분석
- 메타 분석

### 2. Clinical Investigation (임상 시험)
- 전망적 연구
- 후향적 연구
- 다기관 연구

### 3. Post-Market Clinical Follow-up (PMCF)
- 시판 후 임상 추적
- 장기적 안전성 모니터링
- 실사용 데이터 수집

### 4. Registry Data (등록소 데이터)
- 질병 등록소
- 기기 등록소
- 실제 데이터

## 임상 평가 보고서 (CER)

### 구성 요소
1. Clinical Evaluation Plan
2. Literature Search Results
3. Clinical Data Analysis
4. Benefit-Risk Analysis
5. Clinical Evaluation Conclusion
