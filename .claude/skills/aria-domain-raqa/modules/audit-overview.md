# Audit Overview (감사 개요)

## 개요

Audit은 의료기기 품질 시스템과 규제 준수를 검증하는 프로세스입니다.

## 감사 유형

### 1. Internal Audit (내부 감사)
- 품질 시스템 감사
- 프로세스 감사
- 제품 감사
- 공급업체 감사

### 2. External Audit (외부 감사)
- Notified Body 감사
- FDA 감사
- MFDS 검사
- 공급업체 감사

### 3. For Cause Audit (원인 감사)
- 문제 발생 시 특별 감사
- Adverse Event 발생
- Recall 발생
- Complaint 증가

## 감사 준비

### 1. 문서 준비
- SOP 및 Work Instruction
- DHF, DMR, DHR
- CAPA Records
- Training Records
- Complaint Files

### 2. 시설 준비
- 생산 시설
- 창고
- 실험실
- 사무실

### 3. 직원 준비
- SOP 교육
- 인터뷰 준비
- 증거 준비

## 감사 절차

### 1. Opening Meeting
- 감사 범위 확인
- 일정 공유
- 질의응답

### 2. Document Review
- 문서 검토
- 기록 확인
- 추적 가능성 확인

### 3. Facility Tour
- 시설 관찰
- 프로세스 확인
- 직원 면담

### 4. Closing Meeting
- 발견사항 공유
- CAPA 계획
- 마감
