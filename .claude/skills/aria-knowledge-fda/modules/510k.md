# FDA 510(k) - Premarket Notification

## Overview

510(k) is a premarket submission made to FDA to demonstrate that a device is "substantially equivalent" to a predicate device.

## Substantial Equivalence

### Definition

A device is substantially equivalent if it has:
1. The same intended use as the predicate device
2. The same technological characteristics
3. Different technological characteristics but does not raise new questions of safety and effectiveness

### Predicate Device Selection

**Criteria:**
- Legally marketed in the US
- Same intended use
- Similar technological characteristics
- Preferably from same manufacturer

**Sources:**
- FDA 510(k) database
- Product code classification
- Manufacturer documentation

## 510(k) Submission Types

### Traditional 510(k)

**Most common type**
- Requires full comparison to predicate
- Includes all performance data
- Timeline: 3-6 months

### Abbreviated 510(k)

**When:**
- Device is a modification of manufacturer's own device
- Relying on guidance documents or special controls
- Streamlined submission

### Special 510(k)

**When:**
- Device changes are well-established
- Established in guidance documents
- Limited scope modifications

## 510(k) Submission Contents

### Required Sections

**Section 1: User Fee Cover Sheet**
- Form FDA 3601
- Payment verification

**Section 2: Indications for Use**
- Intended use statement
- Contraindications

**Section 3: 510(k) Statement**
- Declaration of substantial equivalence
- Predicate device identification

**Section 4: Device Description**
- Principles of operation
- Specifications and features

**Section 5: Truthful and Accuracy Statement**
- Certification of accuracy
- Signed by authorized official

**Section 6: Class III Summary**
- For class III devices requiring premarket approval

**Section 7: Financial Certifications**
- Small business eligibility

**Section 8: Declarations**
- Compliance declarations

**Section 9: Class II Special Controls**
- Summary of special controls

**Section 10: Proposed Labeling**
- Labeling draft
- Instructions for use

**Section 11: Sterility and Pyrogenicity**
- If applicable to device

**Section 12: Biocompatibility**
- If applicable to device

**Section 13: 510(k) Summary**
- Summary of safety and effectiveness

**Section 14: Attachments**
- Additional supporting information

## Refuse to Accept (RTA) Criteria

### Common RTA Triggers

- Missing required sections
- Incomplete user fee payment
- Missing truthful and accuracy statement
- Inadequate device description
- Missing predicate device identification
- Missing proposed labeling

### RTA Timeline

- FDA has 15 days to review for RTA
- If RTA issued, submitter has 180 days to correct

### Small Business Qualification

**Criteria:**
- Gross revenue < $100 million
- Not part of larger organization
- Submit FDA Form 3602A
