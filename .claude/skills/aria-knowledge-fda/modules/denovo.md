# De Novo Classification

## 개요

De Novo는 새로운 저~중등도 위험 기기에 대한 새로운 Class I 또는 Class II 분류를 요청하는 절차입니다.

## De Novo 적용 조건

### 1. Predicate Device가 없음
- 신규 기기
- 기존 기기와 차별화

### 2. 저~중등도 위험
- Low to Moderate Risk
- Class I 또는 Class II 적합

### 3. 일반 통제 또는 특별 통제로 관리 가능
- General Controls 충분
- Special Controls로 관리 가능

## De Novo 절차

### 1. Pre-Submission (선택)
- FDA와 사전 미팅
- 데이터 요구사항 확인
- 개발 방향 논의

### 2. De Novo Submission
- De Novo 신청서 제출
- 기술 문서
- 안전성·유효성 데이터

### 3. FDA Review (120일)
- 문서 검토
- 안전성 평가
- 분류 결정

### 4. Decision
- 승인: 새로운 Class I/II 생성
- 거부: PMA 제출 권장
- 추가 정보 요청

## De Novo 제출 요소

### 1. Device Description
- 기기 설명
- 작동 원리
- 구성 요소

### 2. Intended Use
- 의도된 사용
- 적응증
- 사용 환경

### 3. Nonclinical Testing
- Bench testing
- Animal studies
- Biocompatibility
- Electrical safety
- EMC

### 4. Clinical Data (필요 시)
- Clinical investigation
- Literature review
- Registry data

### 5. Proposed Labeling
- Labeling draft
- IFU draft
- Warnings and precautions

### 6. Risk Analysis
- Risk management file
- Benefit-risk analysis
- Residual risk

## De Novo vs. 510(k) vs. PMA

| 구분 | De Novo | 510(k) | PMA |
|------|---------|--------|-----|
| 대상 | 신규 저~중위험 | Predicate 있는 기기 | 고위험 기기 |
| 분류 | Class I/II | Class I/II/III | Class III |
| 임상 데이터 | 선택적 | 선택적 | 필수 |
| 심사 기간 | 120일 | 90일 | 180일 |
| 비용 | 중간 | 낮음 | 높음 |
